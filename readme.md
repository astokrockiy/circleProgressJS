Библиотека CircleProgressJS предназначена для создания круговых прогрессбаров через canvas.

## Как использовать

```javascript
circleProgressJS(element);
```

где element - любой элемент в DOM структуре.

## Параметры

Все параметры передаются через data-* свойства передоваемого элемента.

| Параметр              | Значение                                                     | Обязательность | Пример                                              |
| --------------------- | ------------------------------------------------------------ | -------------- | --------------------------------------------------- |
| data-value            | Конечное значение от 0 до 1                                  | Да             | 0.8                                                 |
| data-start-value      | Начальное значение от 0 до 1                                 | Нет            | 0.3                                                 |
| data-diameter         | Диаметр круга прогрессбара в пикселях                        | Да             | 300                                                 |
| data-time             | Время, за которое проходит анимация в милисекундах. Если 0, то прогрессбар строится без анимации | Да             | 1000                                                |
| data-round            | Округляет края, при значении true                            | Нет            | true                                                |
| data-width            | Толщина задней части прогрессбара в пикселях, возможна запись двух значений через ; - тогда эта толщина будет изменяться в процессе. Если не задано - задняя часть не будет отрисовываться | Нет            | 5<br />5;10                                         |
| data-progress-width   | Толщина изменяемой части прогрессбара в пикселях, возможна запись двух значений через ; - тогда эта толщина будет изменяться в процессе | Да             | 10<br />10;20                                       |
| data-color            | Цвет задней части прогрессбара. Если не задано - задняя часть не будет отрисовываться. Цвет может задаваться через #, rgb, rgba. Возможна запись двух значений через ; - тогда цвет будет плавно изменяться | Нет            | #0681c4<br /><br />rgba(255,0,0,0.5);rgb(0,255,0,0) |
|                       | Возможно задать градиент. Для этого необходима начать со слова gradient. Далее в круглых скобках угол в градусов, ";", далее в квадратных скобках черех ; записывается опять в квадратных скобках через ; цвет и значение от 0 до 1 |                | gradient(45;[[#0681c4; 0.5]; [#4ac5f8; 0.5]])       |
|                       | Возможно задать изображение. Для этого необходима начать со слова image. Затем в круглых скобках путь к изображению. |                | image(1.png)                                        |
| data-progress-color   | Цвет изменяемой части прогрессбара. Цвет может задаваться через #, rgb, rgba. Возможна запись двух значений через ; - тогда цвет будет плавно изменяться | Да             | #0681c4<br /><br />rgba(255,0,0,0.5);rgb(0,255,0,0) |
|                       | Возможно задать градиент. Для этого необходима начать со слова gradient. Далее в круглых скобках угол в градусов, ";", далее в квадратных скобках черех ; записывается опять в квадратных скобках через ; цвет и значение от 0 до 1 |                | gradient(45;[[#0681c4; 0.5]; [#4ac5f8; 0.5]])       |
|                       | Возможно задать изображение. Для этого необходима начать со слова image. Затем в круглых скобках путь к изображению. |                | image(1.png)                                        |
| data-shift            | Сдвиг началапрогрессбара в градусах от 0 до 360              | Нет            | 180                                                 |
| data-change-mode-full | При значении true, аниммация будет длится не как указано в data-time. Значение с data-time задаётся для полного изменения от 0 до 1 | Нет            | true                                                |
| data-reverse          | При значении true, значене прогрессбара будет уменьшаться    | Нет            | true                                                |
| data-half-circle      | При значении true, задняя часть прогрессбара не является кругом, а только видимой частью | Нет            | true                                                |
| data-two-sided        | При значении true, прогрессбар растёт с обоих сторон         | Нет            | true                                                |
| data-empty            | При значении true, внутри прогрессбара ничего не пишется     | Нет            | true                                                |
| data-replace          | Заменяет одни символы во внутреннем тексте. Пишется в квадратных скобках через ;. Первое - символ который будет заменяться, второе - символ на который будет заменяться | Нет            | ['.';','];['0';'o']                                 |
| data-change           | Принимает функцию, которая будет обрабатывать текст внутри прогрессбара. Такая функция должна принимать 1 аргумент - текущее значение внутри прогрессбара, возвращает то, что вывести во внутрь прогрессбара | Нет            | changeText                                          |
| data-fixed            | До какой цифры после запятой округлять внутреннее значение. При 0 - округляет до круглых чисел | Да             | 1                                                   |
| data-power            | Во сколько умножить "родное" значение прогрессбара. Изначально оно от 0 до 1 | Да             | 100                                                 |
| data-easing           | Принимает функцию сглаживания анимации. Если отсутсвует анимация будет линейной. Во вложеном файле easing.js есть много примеров таких функций, в том числе и cubic bezier. | Нет            | easeInOut                                           |
